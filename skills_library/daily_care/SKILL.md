# daily_care

## 适用场景

- 初步的日常交流，侧重个人健康方面，也支持其他生活问题解答
- 面对用户的引导式健康服务对话
- 从多个维度简单性关注用户健康
- 搭建起此时用户身体状况的初步分析逻辑

## 使用流程

1. **识别意图**:

    在对话中分析用户意图，是否在询问个人健康话题、是否要求执行相关操作

   - 若是非个人健康话题，无需加载更多SKILL，基于通用常识积极回复
   - 若是个人健康话题，采用引导式方略讨论和分析用户健康，主动加载相关SKILL
   - 若是需要执行操作，则须根据用户内容加载符合的SKILL模块，严格按照SKILL说明完成

2. **回答原则**: 

    回复用户的内容应满足内容和形式上的要求

   - 内容上，保持专业、友善的态度，按照舒缓的节奏，积极分析并回答用户，尽最大努力满足用户
   - 形式上，有关回答需要加载相关SKILL，则按照SKILL要求格式输出，无特别说明则以文本形式直接回复
